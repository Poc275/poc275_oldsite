C# Information Tracker Application

This application is a simple Windows forms app that stores personal information (passwords, shopping lists, credit card details etc.). An interface which defines the information that can be stored was supplied, along with a brief outline of what each method and property needs to do.

                        
public interface IInfoCard
{
    // The name is used to identify the Info Card in the tree view
    string Name { get; set; }

    // This is used to determine where the card is positioned in the tree view
    string Category { get; }

    // Returns the data stored in the info card so it can be written to a file
    string GetDataAsString();

    // This method displays a windows form that displays the data in the info card
    void DisplayData(Panel displayPanel);

    // This method should close the form that is being displayed
    void CloseDisplay();

    // This method displays a modal dialog box that allows the data to be edited.
    // returns true if the data has been changed, false if it has not
    bool EditData();
}

This enabled the application to be extended by implementing the different types of personal information as plug-in modules. This also proved to be an excellent enabler for team working as it allowed multiple people to work on the same application. The interface acted as a ‘contract’, forcing the classes to implement certain methods and properties, meaning that they could be added seamlessly into the main application without any errors.
